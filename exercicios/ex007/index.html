Negrito / Destaque

Deixando o texto em Negrito usando a Tag B(não semântico)

Deixando o texto em Destaque usando a Tag STRONG(semântica)

Itálico / Êfase

Deixando o texto em Itálico usando a tag I(não semantico)

Deixando o texto em Itálico usando a tag EM(semântica)

Marcando os textos

Podemos utilizar um marca texto utilizando a tag Mark

Tag SUP e Tag SUB

Para inserir coisas do tipo x20+3 usamos a tag SUP

Para inserir coisas do tipo H2O usamos a tag SUB

Inserindo items

Para inserir textos usamos a tag INS (semântica)

Para inserir textos também podemos usar a tag U (não semântica)

Texto Deletado

Para deletarmos um texto basta inserir a tag DEL
